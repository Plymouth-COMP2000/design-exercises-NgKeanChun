&nbsp;   # Restaurant Management Application



&nbsp;   This is an Android application developed for the Software Engineering 2 module.



&nbsp;   ## Third-Party Resources Used



&nbsp;   - \*\*Material Components for Android:\*\* For modern UI elements like cards, buttons, and text fields.

&nbsp;     - `com.google.android.material:material:1.13.0`

&nbsp;   - \*\*Retrofit:\*\* For making network requests to a REST API.

&nbsp;     - `com.squareup.retrofit2:retrofit:2.9.0`

&nbsp;   - \*\*Gson Converter:\*\* For serializing and deserializing JSON data with Retrofit.

&nbsp;     - `com.squareup.retrofit2:converter-gson:2.9.0`

&nbsp;   - \*\*AndroidX Navigation Component:\*\* For handling in-app navigation between fragments.

&nbsp;     - `androidx.navigation:navigation-fragment:2.7.7`

&nbsp;     - `androidx.navigation:navigation-ui:2.7.7`

&nbsp;   - \*\*AndroidX Lifecycle Components:\*\* For managing UI component lifecycles and persisting state.

&nbsp;     - `androidx.lifecycle:lifecycle-viewmodel:2.8.3`

&nbsp;     - `androidx.lifecycle:lifecycle-livedata:2.8.3`
